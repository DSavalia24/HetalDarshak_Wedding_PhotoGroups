Wedding Photo Group Website - Direct Link Coordinator Fix

This version makes the coordinator button a normal link to the exact Apps Script URL that worked in your browser.
It opens the Apps Script URL in a hidden iframe and advances the coordinator UI locally.

Files included:
- index.html
- tracker.html
- style.css
- floral-left.png
- floral-right.png
- Code.gs
- README.txt

Add these yourself:
- guests.csv
- image1.jpg

Setup:
1. Use the /exec Apps Script URL, not /dev.
2. Replace PASTE_YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL_HERE in BOTH index.html and tracker.html.
3. Push files to GitHub Pages.
4. Hard refresh tracker.html with Ctrl + F5.
